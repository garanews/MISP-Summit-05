And the winner is...

PostgreSQL with asyncpg has good results: ~ 1M rows/sec

asyncpg is a new fully-featured open-source Python client library for PostgreSQL.
It is built specifically for asyncio and Python 3.5 async / await.
asyncpg is the fastest driver among common Python, NodeJS and Go implementations.
